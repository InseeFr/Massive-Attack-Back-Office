Pour utiliser le swagger operation vous devez avoir

- un fichier contenant le questionnaire au format JSON-Lunatic
- un fichier contenant la description de l'opération (format .fods)
- une archive au format .zip contenant les fichiers json référencés dans le fichier fods.
(ces fichiers sont référencés dans le fichier précédent .fods)


Note : le nom des fichiers n'a aucune importance.
Veuillez ne pas modifier la structure du fichier .fods.

Pour utilisez les fichiers pour l'exemple, vous devez simplement créer une archive contenant les fichiers .json 
et utiliser le swagger avec comme fichiers :
- operation : operation.fods
- questionnaire : questionnaire.xml
- dataZip : l'archive que vous aurez créé
